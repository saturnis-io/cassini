Cassini Showcase Guide

Step-by-step instructions for demonstrating every Analytics feature using the seeded showcase database.
Covers Correlation, Multivariate T2, Predictions, AI Insights, and DOE.

0 Setup

Seed the Database

cd backend
python scripts/seed_showcase.py --force

This creates showcase.db with 3 plants, 36 characteristics, ~18,000 samples, correlated data groups, pre-seeded predictions, DOE studies, and AI insights.

Start the Server

# Use the showcase DB
set CASSINI_DATABASE_URL=sqlite:///showcase.db
cd backend && uvicorn cassini.main:app --reload

9 Technical Notes

Correlation Algorithm

The seed uses Cholesky conditioning to generate correlated data. For each sibling characteristic, values are derived from the anchor using:

z_sibling = r * z_anchor + sqrt(1 - r²) * z_independent

where z_anchor = (value - mean) / std is the standardized anchor value. This preserves the anchor's narrative arc while introducing realistic correlation.

Timestamp Alignment

The multivariate alignment algorithm matches samples across characteristics using a 1-hour tolerance window. Correlated groups use identical timestamps (generated from the anchor characteristic's timestamps), ensuring 100% alignment.
